# Contexto Técnico: verifica-nomes-diarios-oficiais

## Ambiente e Infraestrutura

- **Ambiente Host:** Windows 10/11
- **Gerenciador de Pacotes:** Python (3.11+) via `pip` e `requirements.txt`
- **Banco de Dados:** SQLite (`results.db`)
- **Bibliotecas Principais:** `streamlit` (UI), `pandas` (manipulação de dados), `requests` (scraping HTTP), `beautifulsoup4` (parsing de HTML), `pdfplumber` (parsing de PDFs)

## Estrutura de Diretórios e Componentes

- `requirements.txt`: Lista de dependências do projeto.
- `.env`: Arquivo de variáveis de ambiente locais (nomes a monitorar e links).
- `database.py`: Funções de persistência e gerenciamento do SQLite.
- `scraper.py`: Mecanismo de crawlers/scrapers para DOU, DO-MS, IFMS, Sanesul, MS Gás e CRBM 1.
- `app.py`: Interface web interativa em Streamlit.


## Diretrizes de Geração de Código e Formatação

1. **Formatos PT-BR:** Números nos cards, tabelas e gráficos devem usar vírgula como separador decimal e ponto como separador de milhar.

## RESTRIÇÃO CRÍTICA PARA ECONOMIA DE TOKENS

1. **PROIBIDO executar comandos:** A IA não deve disparar comandos ou terminais de forma automática.
2. **Apenas texto de comando:** Forneça estritamente os comandos limpos em blocos de código markdown para que o desenvolvedor execute manualmente no PowerShell/Terminal do Windows/WSL.
3. **Respostas Concisas:** Sem introduções ou explicações longas. Vá direto ao ponto, priorizando códigos limpos e dados estruturados.
